About the Project

This is the first project from the Udemy Course "Complete C# Unity 2D Game Development", where I created An delivery type of game where the player has to drive a car to deliver packages while avoiding bump into the environment and collect boost power ups. Learning wise, I didn't learn anything new as I already knew the basics of Unity and C#, but it was good for refreshing my memory. I changed the way of some things from the original project like:

  • Used the new Input System instead of the old one.
  • Made some sprites
  • Made a Road Tilemap instead of using individual GameObjects for the road.
