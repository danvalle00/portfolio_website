View Source Code

About the Project

This prototype was one of my first attempts at crafting a metroidvania-style game in Unity, I learned about the new/actual input system, Cinemachine for camera control, and tilemaps with autotiling rules. The game has simple upgrades, like dashing, double jumping. The player movement is inspired by games like Celeste and Hollow Knight. He has dash, double jump, wall slides, wall jumps and poggos.
