About the Project

Pongers is a re-imagining of the classic Pong game, developed as my first complete project in Unity. The goal was to understand the core loop of game development, from input handling to physics collisions and UI management.

Unlike the original, this version features power-ups, dynamic difficulty adjustment, and a localized multiplayer mode.

Key Features

  • Local Multiplayer (1v1)
  • Dynamic ball speed adjustment
  • Retro CRT shader effects
  • Sound effects made with Bfxr

Technical Challenges

One of the main challenges was calculating the bounce angle relative to where the ball hit the paddle. I solved this by mapping the collision point to a normalized range and adjusting the reflection vector accordingly.
