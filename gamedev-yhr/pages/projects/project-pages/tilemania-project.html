About the Project

This is the third project from the Udemy Course "Complete C# Unity 2D Game Development", where I created an very standard platformer. Learning wise, I diverged from the course, using another workflows for inputs (Using Actions) and I was thinking to use Raycasts for ground detections instead of colliders and I thought to use some object pooling logic for the shooting but the project is very basic so I used the standard Instantiate/Destroy methods.
