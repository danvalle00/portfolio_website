About the Project

This is the second project from the Udemy Course "Complete C# Unity 2D Game Development", where I created an slope surfing game with a speed and torque power ups and score logic per flip made.Learning wise, I discovered a few new components of Unity such as Level Sprite Shape and Surface Effector 2D, which were used to create the slope and add movement to it respectively.
